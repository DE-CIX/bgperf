bgperf
========

bgperf is a performance measurement tool for BGP implementation.

## Prerequisites

* Python 2.7 or later
* Docker 1.9.0 or later

## How to install

```bash
$ git clone https://github.com/osrg/bgperf
$ cd bgperf
$ pip install -r pip-requirements.txt
$ ./bgperf.py doctor
docker version ... ok (1.9.1)
bgperf image ... ok
gobgp image ... ok
bird image ... ok
quagga image ... ok
```
